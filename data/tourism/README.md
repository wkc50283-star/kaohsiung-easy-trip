# Tourism Data Layout

本資料夾預留給「一路玩高雄」未來整理官方觀光資料使用，目前只建立資料架構、來源說明與匯入位置，不放大量資料。

## 資料來源原則

- 本站優先使用交通部觀光署觀光資訊資料庫 V2.1 JSON。
- 高雄在地資料優先補充高雄市政府開放資料。
- taiwan.net.tw 僅作官方內容確認與多語參考。
- V1.0 格式不作為新功能主格式。
- 本站資料僅供旅遊規劃參考，實際資訊以官方公告為準。
- 使用政府開放資料時需標示來源。

## 不直接爬取 taiwan.net.tw

taiwan.net.tw 官方網站適合人工確認官方介紹、分類與多語內容，但不作為本站主要爬蟲來源。後續資料匯入應優先使用官方開放資料格式與可追蹤的來源檔。

## 更新方式

第一階段只保留 schema、來源清單與 placeholder 檔案。待交通部觀光署觀光資訊資料庫 V2.1 或後續版本確認後，再匯入正式高雄景點、活動、交通與熱門程度資料。
