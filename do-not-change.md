# do-not-change.md

本文件記錄「一路玩高雄 / 高雄不開車自由行助手」目前已定版或已驗證的禁改清單。

若任務內容與本文件衝突，必須停止並回報，不得直接修改。
若確實需要修改本文件列出的項目，必須取得董事長明確批准。

## 1. 全站版型保護

1. 手機版與電腦網頁版都列入保護。
2. 修改後必須同時檢查手機版與電腦網頁版。
3. 不得為了美化而重排首頁。
4. 不得為了美化而重構 HTML / CSS。
5. 不得順手整理整份 CSS。
6. 不得順手整理格式。
7. 不得修改與任務無關的檔案。
8. 已驗證功能不得動。

## 2. 首頁流程保護

已定版首頁流程不得亂改、刪除或重排。

受保護區塊包含：

1. 品牌頁首。
2. 語言切換。
3. 公告列。
4. Hero 主標與副標。
5. 行程產生卡。
6. 熱門不開車玩法。
7. 更多順路玩法。
8. 旅人實走提醒。
9. Footer。

## 3. 品牌頁首與語言入口

以下項目不得因無關任務修改：

1. 品牌頁首。
2. 英文副標。
3. 語言切換入口。
4. 多語入口。

多語入口不得移除。

## 4. 公告列保護

公告列不得因無關任務亂改。

不得任意：

1. 刪除公告列。
2. 改成警告框風格。
3. 改成紅色或過度刺眼樣式。
4. 破壞手機版閱讀。
5. 影響 Hero 主標與行程產生卡。

## 5. Hero 文案保護

Hero 主標不得因無關任務修改：

> 高雄不開車怎麼玩？

Hero 副標不得因無關任務修改：

> 搭捷運、輕軌、渡輪，把高雄景點順路排好。

## 6. 行程產生卡保護

行程產生卡不得亂改。

受保護內容包含：

1. 出發地選擇。
2. 旅遊天數選擇。
3. 產生按鈕。
4. 已驗證的推薦結果流程。

不得因無關任務修改 planner 主流程。

## 7. 熱門不開車玩法保護

「熱門不開車玩法」區塊不得刪除或重排。

以下熱門卡片不得亂改：

1. 旗津輕旅行。
2. 鹽埕駁二半日。
3. 西子灣夕陽散步。

## 8. 更多順路玩法保護

「更多順路玩法」區塊不得刪除或重排。

以下卡片不得亂改：

1. 避雷提醒。
2. 住宿基地。
3. 順路吃喝。
4. 主景點順路玩法。
5. 雨天／熱天／本地玩法。

## 9. 旅人實走提醒保護

旅人實走提醒區塊不得刪除。

旅人留言、交通、住宿、避雷提醒等既有定位不得混改。

## 10. Footer 保護

Footer 內容不得因無關任務亂改。

受保護項目包含：

1. Footer 說明。
2. 關於本站。
3. 資料來源。
4. Sitemap。

## 11. 路線與路線卡規則

Hero 左營到旗津只能稱為範例路線。
不得寫成固定答案、唯一路線或所有人都適用的推薦。

路線卡不寫時間。
路線卡只寫交通方式與順序。

不得新增幾分鐘、約多久、固定移動時間等表述，除非董事長明確批准。

## 12. 禁止的順手修改

不得因無關任務執行以下行為：

1. 重構。
2. 重排首頁。
3. 整理格式。
4. 整理整份 CSS。
5. 改無關檔案。
6. 改已驗證功能。
7. 改網站功能。
8. 混改旅人留言、交通、住宿、避雷提醒等既有定位。

## 13. 衝突處理

如果任務要求與本文件衝突：

1. 先停止。
2. 明確回報衝突點。
3. 等董事長批准。
4. 未批准前不得修改。
