# P0-1 旗津／鼓山渡輪官方資料查證報告 v0.1

本文件整理備援模式 P0-1「旗津／鼓山渡輪」官方資料查證結果。

本文件不是功能資料，不是 JSON，不接 API，不查即時班次，不寫固定船班、固定班距或末班時間。

## 1. 文件目的

旗津／鼓山渡輪是備援模式第一版的 P0 交通與風險項目。

原因：

1. 旗津屬於海邊與渡輪旅遊圈。
2. 使用者人在旗津時，回收方向高度依賴渡輪與天候。
3. 渡輪營運可能受天候、現場調度、公告與航線服務狀態影響。
4. 若錯誤寫死船班、班距或末班，可能讓旅人越走越遠、錯過回程或誤判現場風險。

本文件目的：

1. 判斷旗津／鼓山渡輪是否可納入備援模式第一版。
2. 區分可安全呈現的內容與仍需 Needs review 的內容。
3. 建立第一版保守文案與分級建議。
4. 避免前台輸出未經官方確認的船班承諾。

## 2. 查證來源

本次只查官方與官方相關入口，不查即時班次，不使用第三方遊記。

官方查證入口：

1. 高雄市輪船股份有限公司官方網站
   - https://kcs.kcg.gov.tw/
2. 高雄市輪船官方航線資訊
   - 查證入口：高雄市輪船官網中的「航線資訊」與「鼓山-旗津航線」相關頁面
3. 高雄市輪船官方輪渡站資訊
   - 查證入口：鼓山輪渡站、旗津輪渡站相關頁面
4. 高雄市輪船官方營運公告
   - 查證入口：高雄市輪船官網中的公告、最新消息或營運公告頁面

查證使用原則：

1. 官方網站可作為航線存在、輪渡站資訊與公告入口的來源。
2. 船班、班距、末班、臨時停航與營運異動不得直接寫死。
3. 若未來要轉成資料，必須再由人工確認官方頁面、公告日期與引用欄位。

## 3. 旗津／鼓山渡輪在備援模式中的風險

| 風險 | 說明 | 備援影響 |
|---|---|---|
| 天候風險 | 雨天、風大、颱風、海象或強降雨可能影響搭乘舒適度與營運狀態 | 不可鼓勵使用者雨天硬排旗津海邊 |
| 現場調度風險 | 渡輪班距與服務可能受現場調度、人潮或公告影響 | 不可寫固定班距或承諾很快有船 |
| 回程風險 | 使用者若時間晚、拖行李或需要回高鐵／住宿，旗津可能增加回收壓力 | 應優先提示先回鼓山或西子灣方向 |
| 體力風險 | 旗津步行、排隊、搭船與海邊移動對長輩、小孩或拖行李者較累 | 應降低行程密度，不建議再多點硬接 |
| 資訊時效風險 | 船班、停航、營運時間可能變動 | 前台只給方向，不給固定班次 |

## 4. 可確認的官方資料

可安全確認的資料類型：

1. 高雄市輪船為官方輪船營運資訊入口。
2. 旗津與鼓山之間存在官方渡輪航線資訊入口。
3. 鼓山輪渡站與旗津輪渡站是該旅遊圈的重要回收點。
4. 官方網站提供航線、站點與公告類資訊入口。
5. 渡輪營運資訊應以高雄市輪船官方公告與現場資訊為準。

可用於備援模式第一版的判斷：

1. 旗津可納入「旗津／西子灣」旅遊圈。
2. 鼓山輪渡站可作為旗津回收方向之一。
3. 西子灣站可作為回到市區的後續回收方向之一。
4. 旗津遇雨天、風大、熱季中午或拖行李時，應保守處理。

## 5. 仍需 Needs review 的資料

以下資料未經當次官方查證前，不得寫進前台確定結果：

| 資料 | Needs review 原因 |
|---|---|
| 固定船班 | 船班可能受公告、現場調度或天候影響 |
| 固定班距 | 班距可能因平假日、人潮、調度或天候而異 |
| 末班時間 | 末班屬高風險承諾，不可人工猜 |
| 是否現在有船 | 第一版不接即時資訊，不可判斷 |
| 是否一定趕得上 | 受位置、排隊、人潮、天候與營運影響 |
| 停航或異動狀態 | 需看官方公告與現場資訊 |
| 船是否可載腳踏車或機車 | 需依官方最新規定確認 |
| 票價與支付方式 | 可能異動，需官方確認 |

## 6. 第一版可安全呈現的說法

第一版可用保守說法：

1. 旗津可作為海邊與渡輪旅遊圈。
2. 若人在旗津，回收方向優先往鼓山輪渡站，再接西子灣或鹽埕／駁二方向。
3. 雨天、風大、熱季中午、拖行李或時間晚時，不建議把旗津當主要行程。
4. 渡輪與船班可能受天候、現場調度與公告影響，請以高雄市輪船官方公告與現場資訊為準。
5. 第一版只提供方向，不提供船班、班距、末班或一定趕得上的承諾。

## 7. 第一版不可寫死的說法

第一版不得寫：

1. 固定船班。
2. 固定班距。
3. 末班時間。
4. 現在一定有船。
5. 一定趕得上。
6. 搭某一班一定可以回市區。
7. 雨天仍適合硬走旗津海邊。
8. 拖行李可以放心進旗津。
9. 旗津可以無條件接回所有市區景點。

## 8. 下雨、風大、熱季、拖行李時的保守提醒

| 情境 | 保守提醒 |
|---|---|
| 下雨 | 不建議硬走旗津海邊。若已在旗津，優先往鼓山輪渡站回收，後續改室內或商圈。 |
| 風大 | 渡輪與海邊體驗可能受影響，先看高雄市輪船官方公告與現場資訊。 |
| 熱季中午 | 不建議長時間走旗津、西子灣與港邊戶外，改早上或傍晚較保守。 |
| 拖行李 | 不建議拖大行李進旗津。若已在旗津，先回鼓山或西子灣方向，再接住宿或車站。 |
| 時間晚 | 不顯示末班，也不承諾趕得上；應先查官方資訊並保守回收。 |

## 9. 使用者人在旗津想繼續玩時的分級建議

| 分級 | 建議方向 | 說明 |
|---|---|---|
| 順路可玩 | 旗津老街周邊、渡輪前後短線 | 只限天氣穩定、體力足夠、時間不晚，且不拖大行李。 |
| 可玩但較繞 | 回鼓山後接西子灣、鹽埕／駁二 | 需要多一次渡輪與步行或轉乘，適合時間較充裕時。 |
| 不建議但可自行決定 | 雨天硬走海邊、熱季中午長時間戶外、最後一天拖行李進旗津 | 風險在於曝曬、淋雨、排隊、回程壓力與交通資訊不確定。 |

## 10. 使用者人在旗津想回收時的安全回收方向

第一版回收方向：

1. 先往旗津輪渡站。
2. 搭渡輪回鼓山方向。
3. 回到鼓山後，優先接西子灣站、鹽埕／駁二或美麗島／高雄車站方向。
4. 若要回左營或高鐵，先回到捷運或台鐵主幹，再判斷，不從旗津硬接多點。

保守提醒：

1. 不寫船班與末班。
2. 不承諾一定趕得上。
3. 若時間晚、雨天、風大或有大行李，應優先回收，不建議再補景點。
4. 實際渡輪、船班與營運狀態以高雄市輪船官方公告與現場資訊為準。

## 11. 第一版建議文案

### 一般狀況

你現在在旗津，建議先把行程收斂在旗津／鼓山渡輪這一圈。天氣穩定、時間足夠時，可以短線走旗津老街周邊；如果想回市區，先往鼓山渡輪方向回收，再接西子灣、鹽埕或美麗島。

### 雨天或風大

雨天或風大時，不建議硬走旗津海邊。建議先往鼓山渡輪方向回收，後續改走室內、商圈或車站周邊。渡輪與船班可能受天候、現場調度與公告影響，請以高雄市輪船官方公告與現場資訊為準。

### 熱季中午

4 到 10 月高雄偏熱，熱季中午不建議長時間走旗津、西子灣或港邊戶外。若已在旗津，建議縮短停留，先回鼓山或西子灣方向，再安排室內休息點。

### 拖行李或最後一天

不建議拖大行李進旗津。若已在旗津，先回鼓山或西子灣方向，再接住宿區、車站或左營回程線，不要再補太多點。

## 13. 王王結論

旗津／鼓山渡輪可以納入備援模式第一版。

但第一版只能用：

1. 旅遊圈。
2. 回收方向。
3. 風險提醒。
4. 官方公告優先。
5. Needs review 標記。

第一版不得使用：

1. 固定船班。
2. 固定班距。
3. 末班時間。
4. 現在一定有船。
5. 一定趕得上。

王王審核結論：

旗津是可以做備援的，但不能做成交通保證。使用者在旗津時，第一句話不是鼓勵他繼續跑點，而是先判斷天氣、時間、體力與行李，再決定要不要回鼓山方向收斂。這樣才符合「不要害旅人越走越遠」。
