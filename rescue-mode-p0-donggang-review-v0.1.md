# P0-3 東港／華僑市場回高雄官方資料查證報告 v0.1

本文件為「一路玩高雄 / 高雄不開車自由行助手」備援模式 P0-3 查證文件。

本文件只做官方與可信資料查證，不是功能施工資料，不是 JSON，不接 API，不查即時班次。

## 1. 文件目的

確認使用者人在東港／華僑市場時，第一版備援模式能不能安全提供「回高雄」方向，以及哪些內容必須保留 Needs review。

核心原則：不要害旅人越走越遠。東港已屬屏東跨縣市回收圈，若錯把它當成高雄市區短線，會造成回程、住宿與體力風險。

## 2. 查證來源

本次查證來源包含：

1. 屏東縣政府與屏東縣交通旅遊相關公開資訊入口：https://www.pthg.gov.tw/
2. 高雄客運官方網站：https://www.ksbus.com.tw/
3. 屏東客運官方網站／屏東客運相關資訊入口。
4. 東港轉運站公開資料與地方交通整理資料。
5. TDX 官方 API 分類資料：https://tdx.transportdata.tw/webapi/serviceCategory/ForSwagger
6. 台灣好行與高墾線相關公開資料。

注意：可信輔助資料可用來判斷查證方向，但不能直接取代官方時刻、站牌、方向與末班資訊。

## 3. 東港／華僑市場在備援模式中的風險

東港／華僑市場是 P0 風險，原因如下：

1. 已離開高雄市區旅遊圈，屬屏東跨縣市回收圈。
2. 可能牽涉東港轉運站、客運、東琉線碼頭、華僑市場與住宿決策。
3. 使用者可能把小琉球船班與回高雄交通混在一起，造成錯誤判斷。
4. 回高雄方向可能有小港、左營、高雄車站或市區不同路徑，不能猜。
5. 若時間晚、天氣差、拖行李或帶長輩小孩，不能再硬接高雄港邊或市區多點。

## 4. 可確認的官方資料

目前可安全確認：

1. 東港轉運站可列為東港地區主要交通回收點候選。
2. 東港／華僑市場已不是高雄市區短線，第一版必須先問今晚住哪裡與是否一定回高雄。
3. 高雄客運與屏東客運是高屏跨區客運資訊的重要查證來源。
4. TDX 有公車／公路客運相關分類，可作為後續第二階段驗證來源。
5. 東琉線、小琉球船班不應直接混入「回高雄」回收建議，除非使用者目標是去小琉球。

## 5. 仍需 Needs review 的資料

以下資料第一版不可寫死，必須標 Needs review：

1. 華僑市場到東港轉運站的實際動線。
2. 東港轉運站回高雄的官方路線。
3. 是否可回捷運小港站。
4. 是否可回高鐵左營站。
5. 是否可回高雄車站或高雄市區。
6. 上下車站牌名稱。
7. 上下車方向。
8. 班距。
9. 末班車。
10. 是否每日營運。
11. 是否受連假、活動、人潮、颱風、豪雨或道路狀況影響。
12. 東琉線碼頭與東港轉運站是否可互為回收點。
13. 小琉球船班是否影響東港當地交通與人潮，但不得混成回高雄建議。

## 6. 第一版可安全呈現的說法

可以寫：

> 你目前在東港／華僑市場一帶，已經屬於屏東跨縣市回收圈。若要回高雄，請先確認今晚住哪裡、是否一定回高雄，再回到東港轉運站或主要交通點查官方班次。

可以寫：

> 東港不適合臨時再硬接高雄市區多個景點。時間晚、天氣不好或拖行李時，優先處理回程或住宿。

可以寫：

> 小琉球船班不是回高雄的備援路線。除非使用者明確要去小琉球，否則第一版不要把船班混入回高雄建議。

## 7. 第一版不可寫死的說法

不要寫：

1. 搭某某路線一定能回高雄。
2. 每幾分鐘一班。
3. 末班是幾點。
4. 從華僑市場走幾分鐘一定到東港轉運站。
5. 一定可直達高鐵左營。
6. 一定可直達高雄車站。
7. 現在一定有車。
8. 一定趕得上。
9. 東港玩完還能順路接旗津、駁二、瑞豐夜市或夢時代。
10. 小琉球船班可以當成回高雄替代路線。

## 8. 天氣、時間晚、拖行李、長輩小孩、體力不足時的保守提醒

### 天氣不好

豪雨、強風、颱風或海邊天候不佳時，不建議再往碼頭或小琉球方向延伸。請先確認官方交通與航運公告。

### 時間晚

時間晚時，不建議再加高雄市區景點。應優先確認東港轉運站回程班次、住宿方向或是否改住屏東／東港。

### 拖行李

拖大行李時，不建議在華僑市場、碼頭與轉運站之間長距離移動。第一版只提醒回主要交通點查官方班次。

### 長輩小孩

長輩、小孩同行時，不建議安排不確定轉乘、不確定等車時間或人潮壅擠路線。

### 體力不足

體力不足時，備援模式不應補景點，應優先回收或找住宿／餐食點。

## 9. 使用者人在東港／華僑市場想繼續玩時的分級建議

### 順路可玩

東港當地短停留、華僑市場周邊、東港市區吃飯。

限制：只適合時間充足、天氣可接受、回程已確認時。

### 可玩但較繞

回到高雄小港、左營、高雄車站或住宿區後，再接附近晚餐或商圈。

限制：不能包裝成順路景點，必須先確認官方交通。

### 不建議但可自行決定

東港再接旗津、駁二、瑞豐夜市、蓮池潭、夢時代或高雄遠距多點。

理由：跨縣市後再跨高雄市區，交通不確定且容易越走越遠。

## 10. 使用者人在東港／華僑市場想回收時的安全回收方向

第一版建議：

> 你目前在東港／華僑市場。若要回高雄，先不要再加高雄市區景點。請優先回到東港轉運站或主要交通點，查屏東客運、高雄客運或官方交通資訊，再決定回小港、左營、高雄車站或住宿區。

可標記的安全回收候選：

1. 東港轉運站
2. 華僑市場周邊主要交通點
3. 捷運小港站方向
4. 高鐵左營站方向
5. 高雄車站／高雄市區方向
6. 今晚住宿地點

但路線編號、站牌、方向、班距、末班仍需官方確認。

### 短版

你目前在東港／華僑市場一帶，已經是屏東跨縣市回收圈。若時間晚、天氣不好或拖行李，先不要再接高雄市區景點。請優先回東港轉運站或主要交通點，確認官方客運班次後，再往小港、左營、高雄車站或住宿方向移動。

順路可玩：

## 13. 王王結論

東港／華僑市場可以納入備援模式第一版，但只能用「跨縣市回收圈、東港轉運站、回高雄方向、風險提醒、官方資訊優先」呈現。

不能寫固定路線、固定站牌、固定班距、固定末班，也不能承諾一定有車或一定趕得上。

第一版最安全做法：

> 人在東港／華僑市場 → 先問今晚住哪裡與是否一定回高雄 → 優先回東港轉運站或主要交通點 → 查官方班次 → 回小港、左營、高雄車站或住宿方向。
