# P0-2 旗山／美濃回高雄官方資料查證報告 v0.1

本文件為「一路玩高雄 / 高雄不開車自由行助手」備援模式 P0-2 查證文件。

本文件只做官方與可信資料查證，不是功能施工資料，不是 JSON，不接 API，不查即時班次。

## 1. 文件目的

確認使用者人在旗山／美濃時，第一版備援模式能不能安全提供「回高雄」方向，以及哪些內容必須保留 Needs review。

核心原則：不要害旅人越走越遠。旗山／美濃屬高雄遠郊文化圈，若客運方向、站點、班距或末班資訊寫錯，會直接影響回收安全。

## 2. 查證來源

本次查證來源包含：

1. 高雄客運官方網站：https://www.ksbus.com.tw/
2. 高雄市公車動態資訊系統：https://ibus.tbkc.gov.tw/
3. 高雄市政府交通局相關公車資訊入口：https://www.tbkc.gov.tw/
4. TDX 官方 API 分類資料：https://tdx.transportdata.tw/webapi/serviceCategory/ForSwagger
5. 可信輔助資料：旗山轉運站、旗美國道快捷、高旗美濃快線等公開整理資料。

注意：可信輔助資料可用來判斷查證方向，但不能直接取代官方時刻、站牌、方向與末班資訊。

## 3. 旗山／美濃在備援模式中的風險

旗山／美濃是 P0 風險，原因如下：

1. 屬高雄遠郊，不是市區捷運圈。
2. 回高雄主要仰賴客運或轉乘，不適合用市區短線邏輯。
3. 旅人若錯過班次、走錯站、搭錯方向，可能被留在遠郊。
4. 熱季、下雨、時間晚、拖行李、長輩小孩同行時，體力與安全風險升高。
5. 路線編號、站牌、上下車方向、班距與末班都不可猜。

## 4. 可確認的官方資料

目前可安全確認：

1. 高雄客運是旗山／美濃往返高雄方向的重要客運業者之一。
2. 旗山轉運站可列為旗山區的主要交通回收點候選。
3. 高客美濃站或美濃主要交通點可列為美濃區的回收點候選。
4. TDX 有公車／客運相關資料分類，可作為未來第二階段驗證來源。
5. 高雄市公車動態資訊系統與高雄客運官方網站應作為班次、路線、站點與營運異常的優先查證入口。

## 5. 仍需 Needs review 的資料

以下資料第一版不可寫死，必須標 Needs review：

1. 旗山轉運站實際可搭回高雄的路線編號。
2. 美濃回高雄的實際路線編號。
3. 是否直回高鐵左營。
4. 是否直回高雄車站。
5. 上下車站牌名稱。
6. 上下車方向。
7. 班距。
8. 末班車。
9. 是否每日營運。
10. 假日、連假或活動期間是否改班。
11. 是否受颱風、豪雨、道路中斷或交通管制影響。
12. 美濃到旗山轉運站之間是否需要先接駁。

## 6. 第一版可安全呈現的說法

可以寫：

> 你目前在旗山／美濃遠郊文化圈。若要回收，請優先回到旗山轉運站或美濃主要交通點，確認官方客運班次後，再往高雄市區、左營或住宿方向移動。

可以寫：

> 旗山／美濃不適合臨時再硬接旗津、駁二、三多、夢時代或其他跨區景點。時間晚或體力不足時，優先處理回程。

可以寫：

> 客運路線、站牌、方向、班距與末班請以高雄客運、高雄市公車動態資訊系統或官方公告為準。

## 7. 第一版不可寫死的說法

不要寫：

1. 搭某某路線一定可以回高雄。
2. 每幾分鐘一班。
3. 末班是幾點。
4. 從美濃一定可以直達高雄車站。
5. 從旗山一定可以直達高鐵左營。
6. 現在一定有車。
7. 一定趕得上。
8. 回高雄後還能順路接旗津、駁二、瑞豐夜市。

## 8. 天氣、時間晚、拖行李、長輩小孩、體力不足時的保守提醒

### 天氣不好

雨勢大、颱風、豪雨或道路狀況不明時，不要建議旅人繼續往更遠郊移動。應優先查官方交通公告並回到主要交通點。

### 時間晚

時間晚時，不建議再加景點。第一版只提醒先回旗山轉運站或美濃主要交通點確認官方班次。

### 拖行李

拖大行李不建議深入遠郊點位。若已在美濃或旗山，優先回主要交通點，再往左營、高雄車站或住宿區移動。

### 長輩小孩

長輩、小孩同行時，不建議安排需要等車很久、轉乘不明或步行過長的回收方式。

### 體力不足

體力不足時，備援模式不應補景點，而是建議回收。

## 9. 使用者人在旗山／美濃想繼續玩時的分級建議

### 順路可玩

旗山老街、旗山周邊短停留、美濃市區短線。

限制：只適合白天、時間足夠、天氣可接受、體力正常時。

### 可玩但較繞

從旗山／美濃回到左營、高雄車站或市區後，再依狀況接晚餐、商圈或住宿附近活動。

限制：不能寫成一定順路，必須提醒先確認官方班次。

### 不建議但可自行決定

旗山／美濃再接旗津、駁二、夢時代、瑞豐夜市、蓮池潭或其他跨區點。

理由：距離拉長，交通不確定，容易越走越遠。

## 10. 使用者人在旗山／美濃想回收時的安全回收方向

第一版建議：

> 你目前在旗山／美濃。若要收回行程，先不要再加遠的景點。請優先回到旗山轉運站或美濃主要交通點，查高雄客運或高雄市公車動態資訊系統的官方班次，再往高雄車站、左營或住宿方向移動。

可標記的安全回收候選：

1. 旗山轉運站
2. 美濃主要交通點／高客美濃站
3. 高鐵左營站
4. 高雄車站
5. 住宿區附近交通主幹

但路線編號、站牌、方向、班距、末班仍需官方確認。

## 11. 第一版建議文案

### 短版

你目前在旗山／美濃遠郊文化圈。若時間晚、天氣不好、拖行李或體力不足，先不要再加景點。請優先回旗山轉運站或美濃主要交通點，確認官方客運班次後，再往高雄市區、左營或住宿方向移動。

### 分級版

你目前在旗山／美濃一帶。

順路可玩：
旗山老街、旗山周邊短停留、美濃市區短線。

可玩但較繞：
先回左營、高雄車站或市區交通主幹，再接住宿附近吃飯或商圈。

不建議但可自行決定：
旗津、駁二、夢時代、瑞豐夜市、蓮池潭。這些方向會拉長交通，不適合臨時補行程。

提醒：
客運路線、站牌、方向、班距與末班請以高雄客運、高雄市公車動態資訊系統或官方公告為準。

## 13. 王王結論

旗山／美濃可以納入備援模式第一版，但只能用「遠郊文化圈、主要交通回收點、回收方向、風險提醒」呈現。

不能寫固定路線、固定站牌、固定班距、固定末班，也不能承諾一定有車或一定趕得上。

第一版最安全做法：

> 人在旗山／美濃 → 先判斷遠郊風險 → 優先回旗山轉運站或美濃主要交通點 → 查官方班次 → 回左營、高雄車站或住宿方向。
